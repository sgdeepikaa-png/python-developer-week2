# python-developer-week2


## Overview
This week focused on Python control flow using conditionals and loops.

## Programs Built

### 1. Number Guessing Game
- Generates a random number between 1 and 100
- Gives hints whether the guess is too high or too low
- Tracks number of attempts

### 2. Grade Classifier
- Accepts multiple student scores
- Assigns grades A–F using if/elif/else
- Displays summary and class average

### 3. Times Table Generator
- Generates a formatted N × N multiplication table
- Uses nested for loops

### 4. Password Validator
- Validates password strength
- Checks length, uppercase letters, digits, and special characters

### 5. VS Code Debugging
- Practiced breakpoints, watches, and stepping through code

## Concepts Learned
- if, elif, else
- for loops
- while loops
- break and continue
- range()
- enumerate()
- nested loops
- input validation
- debugging in VS Code
